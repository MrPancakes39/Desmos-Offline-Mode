## About

Desmos Offline Mode is a desktop-focused app for using the Desmos calculator experience offline.

The goal is to download the required Desmos resources once while you are online, store them locally, and then load them from your computer on later starts. The desktop version wraps that local app in a normal desktop window and can add native features such as file dialogs, menus, and update UI.

DSOM is the surrounding app and resource-management project. It is not Desmos, and it does not own Desmos calculator code, assets, trademarks, or services.
